This plugin will NOT work for a multisite/network WordPress install - intended for single WordPress ONLY.

== Installation ==

1. Upload the full directory into your wp-content/plugins directory
2. Activate the plugin at the plugin administration page
3. Enter email domains under WP-Admin -> BuddyPress -> Restrict Email
